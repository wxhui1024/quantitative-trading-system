# A股量化选股系统

基于涨停启动后缩量回调策略的自动选股工具

## 功能特性

- 自动筛选满足"涨停启动后缩量回调"形态的股票
- 数据来源：akshare或其他高质量数据源
- 生成Obsidian友好的Markdown表格
- 自动推送结果到GitHub

## 策略说明

- 筛选范围：沪深主板 + 创业板
- 排除：科创板、北交所、ST股票
- 市值范围：50亿-500亿人民币
- 形态识别：考察T-3至T-10交易日内的涨停及成交量模式

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

```bash
python main.py
```
